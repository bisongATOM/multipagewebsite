My Multipage Website

This is a responsive multipage website built using HTML5, CSS3, and JavaScript. It demonstrates fundamental web development skills.

Features

*   Responsive Design: Adapts to different screen sizes (desktop, tablet, mobile).
*   Multiple Pages: Includes Home, About, and Contact pages with consistent navigation.
*   Interactive Elements: Includes form validation using JavaScript.
*   Semantic HTML:Uses HTML5 tags for better structure and accessibility.

Pages

*   Home (`index.html`): Introduction and overview.
*   About (`about.html`): Information about the site or author.
*   Contact (`contact.html`):Contact form with validation.

Technologies Used

*   HTML5
*   CSS3 (Flexbox/Grid, Media Queries)
*   JavaScript (DOM Manipulation, Form Validation)
